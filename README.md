# Angular Storybook

Example on how to use Angular & Storybook together when developing UI Components.  

## Published Link

Application is published in Netlify.  
Link: https://angular-storybook.netlify.app/

## Environment

Angular 10.1.2  
Storybook 6.0.21  
Typescript 4.0.2  

## Installation

```
npm install
```

## Runing Storybook

```
npm run storybook
```

## Storybook Build

```
npm run build-storybook
```
